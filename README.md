# Bondify

**Bondify is an innovative platform designed to foster meaningful connections between individuals, enabling them to network, collaborate, and grow both personally and professionally. Through Bondify, users can seamlessly connect with like-minded people and expand their social and professional circles.**

## Important Links

[Live-Link](https://bondify-48743.web.app)

[Client-Repository](https://github.com/Naimul9/Bondify_client)

[Server-Repository](https://github.com/Naimul9/Bondify_server)


## Features 

- **Search:** A user can search to find specific user. 
- **JWT:** The browser generates a unique token for each user upon login, ensuring secure and individualized session management throughout their interaction with the platform. 
- **Responsive Design:** Enjoy a seamless experience across various devices, including desktops, tablets, and smartphones.

## Steps To Run in Local
- First clone the repo in your local computer using git clone or download the zip file.
- Then install all the dependencies using **npm i**.
- Then set the firebase and mongodb credentials to run the project.
